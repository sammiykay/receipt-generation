Offline Receipt Generator

This project is now app-directory based.
Run all commands from this folder.

Run options:

1) Recommended:
   python run.py

2) Alternative:
   python main.py

Build options:

- Linux/macOS:
  ./build.sh

- Windows PowerShell:
  ./build_windows.ps1

The app starts on 127.0.0.1 using an available port and opens the default browser.
